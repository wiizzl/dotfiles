# dotfiles

My dotfiles, managed with [`chezmoi`](https://github.com/twpayne/chezmoi).

Used to maintain my devices across different environments:

- Arch Linux (minimal)
- macOS

## Required

- [chezmoi](https://www.chezmoi.io/install)

## Installation

1. Init my dotfiles

```sh
chezmoi init wiizzl
```

2. Apply them

```sh
chezmoi apply
```
